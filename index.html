Problem Statement

User Stories

    As a participating art walk gallery owner

  • I want my shop to appear on a map so that people can find us.

    As an art walk visitor

  • I want to see a map of participating galleries, so that I know where I can expect to see art.
  • I want to see some details about the gallery, so I can decide if I want to go there.
  • I want to know my current location, so that I have a point of reference for what galleries I'm near.

    As developers

  • We want to gather all the data we need to build out the data of participating art walk galleries.
  • We want to adopt the visual brand of the current PioneerSquare.org website, so that there is a consistent experience for users.
  • We want to use Google Maps API so that our map is familiar to users.
  • We want to make this as mobile responsive as possible, so that users can access this on their mobile devices.
